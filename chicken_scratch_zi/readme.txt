Made as an extra for Ghost Jam 2026.

This balloon is strongly keyed to the ghost Thoughtful Replies.
- It only has one balloon size.
- Each balloon pattern comes in 3 versions, one for each of the fonts the balloon supports.
- The script input box uses Courier New to help developers, while the other default input boxes use the middle font, AnkeCalligraphic. There are extra input boxes provided so that you can select any of the fonts with a bit of logic on the ghost side.
- It also has some extra balloon surfaces that aren't meant for writing text on. (They can display a few brief words and nothing more.)

Despite that, if you find it useful, feel free to use it per my usual balloon permissions!
